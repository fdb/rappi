Broadcasts

Fetch a list of current broadcasts on Meerkat. The url can be used to open a Meerkat window.

https://rappi.herokuapp.com/meerkat/broadcasts.json

Example

$.getJSON('https://rappi.herokuapp.com/meerkat/broadcasts.json', function(result) {
  if (result.status !== 'ok') {
    alert(result.message);
  }
  // Show the first broadcast in an external window.
  var b = result.broadcasts[0];
  window.open(b.url, '_blank', 'width=300,height=600');
});
